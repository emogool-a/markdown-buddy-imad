# Reflection

## 1. What challenges did you face using AI to generate Markdown content?
One challenge I faced was understanding how Markdown formatting works, especially for code blocks and headings. The AI-generated content sometimes included extra instructions that were not part of the final document, so I had to carefully review and clean the output.

## 2. How did you verify or improve the AI-generated Markdown?
I reviewed the content manually and removed unnecessary instructions. I also checked that the formatting displayed correctly on GitHub and ensured that headings, lists, and code sections were properly structured.

## 3. What are the advantages and limitations of using AI for documentation?
The advantage of using AI is that it quickly generates structured and organized content, saving time. However, the limitation is that the output may include irrelevant or incorrect information, so it requires human review and editing before final use.
